# disk-locking
Este projeto apresenta a solução de disk-locking apresentado na literatura.

boreas4.pro é a subrotina disponibilizada por Cranmer & Saars (2011) para o cálculo da luminosidade, taxa de perda de massa e vento estelar.

O disk-locking é um mecanismo em que a rotação de uma estrela jovem é regulada pela interação magnética com seu disco de acreção. Nesse processo, o campo magnético da estrela conecta-se ao disco e transfere momento angular, impedindo que a estrela acelere sua rotação enquanto ainda está acumulando matéria. Isso ajuda a explicar por que muitas estrelas jovens giram mais lentamente do que o esperado durante suas fases iniciais de evolução.
